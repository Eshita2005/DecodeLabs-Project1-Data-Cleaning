# Project 1 - Data Cleaning & Preparation

## Overview

This project focuses on cleaning and preparing a raw dataset for further analysis. The objective was to identify and handle missing values, check for duplicate records, validate data formats, and create a clean dataset ready for analysis.

## Tools Used

* Python
* Pandas
* Visual Studio Code

## Tasks Performed

1. Loaded the dataset using Pandas.
2. Identified missing values in the dataset.
3. Replaced missing values in the CouponCode column with "No Coupon".
4. Checked for duplicate rows and duplicate IDs.
5. Verified date formats and corrected inconsistencies if any.
6. Generated a cleaned dataset for future analysis.

## Results

* Missing values identified and handled successfully.
* Duplicate records checked and removed if present.
* Date formats validated.
* Cleaned dataset created and saved as `Cleaned_Dataset.xlsx`.

## Files Included

* `project1.py` – Python script for data cleaning.
* `Cleaned_Dataset.xlsx` – Cleaned dataset.
* `Screenshots` – Output screenshots and proof of execution.

## Outcome

The dataset was successfully cleaned and prepared for further analysis, ensuring data quality and reliability for future projects.
